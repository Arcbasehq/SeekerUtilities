# Security Policy

## Supported Versions

We accept security reports for the latest release and the current development branch.

## Reporting a Vulnerability

Please report security issues privately by emailing:

- support@arcbase.one

Include as much detail as possible:

- A clear description of the issue
- Steps to reproduce
- Affected versions and environment
- Proof-of-concept code, logs, or screenshots (if applicable)

We aim to acknowledge reports within 72 hours and will provide updates as we investigate.

## Coordinated Disclosure

Please do not publicly disclose issues until we have provided a fix or mitigation.

## Security Notes

- This project runs privileged system commands for certain maintenance tasks via polkit.
- Do not run the app on systems you do not trust or in shared environments without review.
- Keep your OS and dependencies up to date.
